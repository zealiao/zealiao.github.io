第一个脚本

^j 代表按键CTRL+J
Send 是命令，Hello！ 是要发送的内容
Return 表示停止代码继续执行

快捷键&热字串

热键就是触发动作的按键，如： ==^J::==
热字串就自动执行替换文本或执行程序的缩写内容，如： ==::sfz1::==
区别在于热字串会在输入完就立马替换和执行

热键的创建必须把组合按键在两个冒号的左边，且以return结束
热键和热字串都能启动程序和执行脚本

窗口特定的热键/热字串

有时候你也许想要热键或热字串只在某些特定窗口上工作(或禁用). 要做到这一点, 您需要使用其中的任意一个”高级”命令, 在它们前面带有一个 #:

IfWinActive后面接窗口标题，指定脚本在特定窗口下生效，如不指定，则对所有窗口生效，且与位置有关，它们将影响脚本中的位于他们之间所有热键和热字串. 它们也相互排斥的; 也就是说, 只有最近的一个会有效.

多个热键和热字串都能放在一个脚本里

命令和函数

命令

Command, 参数1, 参数2, 参数3

对象

对象是一种组织数据以实现更高效使用的方法. 有时候对象被当作数组
对象基本上是变量的集合. 变量的名称叫做”键”, 变量的内容叫做”值”.
![[Pasted image 20201212143137.png]]
列表创建：用方括号

MyObject := ["one", "two", "three", 17]

字典创建：用花括号

Banana := {"Color": "Yellow", "Taste": "Delicious", "Price": 3}

数组函数创建：用圆括号 Array函数名

MyObject := Array("one", "two", "three", 17)

对象函数创建：用圆括号 Object函数名

Banana := Object("Color", "Yellow", "Taste", "Delicious", "Price", 3)
